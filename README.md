# 📘 Sistema de Gamificação para Plataforma de Educação Continuada

Este projeto consiste em um sistema educacional completo com três ambientes principais — **Aluno**, **Professor** e **Curador** — cada um com páginas próprias, design premium e interfaces responsivas. A aplicação é estruturada em HTML e CSS puro, com foco em usabilidade, padronização visual e modularidade.

---

# 🎨 Identidade Visual

Todo o projeto segue um **Design System Premium**, composto por:

- Paleta azul-navy + azul-claro  
- Sombras suaves (soft shadow)  
- Cards elevados  
- Botões *pill*  
- Tipografia: **Inter**  
- Headers com gradiente  
- Layouts modulares em grid  
- Navegação fixa superior

---

# 🧩 Funcionalidades por Módulo

## 👨‍🎓 Módulo do Aluno
Páginas disponíveis:
- **Sala de aula** (player, timeline, comentários, materiais)
- **Conquistas** (badges, progressos, conquistas visuais)
- **Catálogo / Cursos / Perfil** *(expansível)*

Destaques:
- Timeline híbrida premium  
- Player estilizado  
- Cards de materiais  
- Sistema de comentários

---

## 👨‍🏫 Módulo do Professor
Páginas já concluídas:

### **📊 Dashboard**
- KPIs (engajamento, presença, avaliações pendentes, aulas no mês)
- Gráficos simulados (linha e barras)
- Atividades recentes

### **📅 Agenda**
- Calendário mensal premium  
- Eventos categorizados (aula, reunião, avaliação, atendimento)
- Sidebar de eventos futuros

### **📝 Avaliações**
- Lista de avaliações  
- Filtros pill com design premium  
- Cards de avaliação com progresso  
- Sessão de avaliações em andamento (cards horizontais)

### **🎥 Aula ao Vivo**
- Player 16:9  
- Controles (mic, câmera, tela, encerrar)  
- Chat ao vivo  
- Lista de participantes  
- Enquete ao vivo

### **📈 Desempenho**
- KPIs da turma  
- Gráfico de linha  
- Gráfico de barras  
- Heatmap semanal  
- Tabela premium com progresso dos alunos

### **📂 Materiais**
- Upload de arquivos  
- Grid com cards de PDF, vídeo, link, slides  
- Ações: baixar, compartilhar, excluir  
- Pastas organizadoras

### **💬 Fórum do Professor**
- Filtros premium (Todos, Metodologia, Planejamento, etc.)
- Cards de tópicos com avatar, tags, contadores e última atividade  
- Sidebar com tópicos populares  
- Botão flutuante “Criar tópico”
